Link: target = '_self'

Opens link in the same tab/window - default behavior. E.g. Click Me

Link: target = '_blank'

Opens link in a new tab/window. E.g. Click Me 

          Best practice (security): Always use rel="noopener noreferrer" when linking external links.
          noopener: This prevents the new page from controlling your page (prevents accessing window.opener).
          noreferrer: Hides the referrer information. The destination site cannot see where the user came from.

          Below example demonstrates how a malicious page can redirect the original page using window.opener when the rel="noopener" attribute is not set.
          (Since, most browsers now a days, add the noopener attribute automatically as part of security, we are explicitly setting rel="opener" to disable the browser's security control for the sake of demonstrating this example.)
